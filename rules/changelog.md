## 0.10.5 Changelog

## Classes

- Added the 5th level Expert Features to all Classes
- Casters gain +1 MP at levels 3, 7 and 10
- Hybrids gained +1 MP at levels 1 and 10
Class Tables

- Level 7 is now Subclass Expert Features
- Level 8 is now a Talent, Path Progression and 2 Ancestry
Points
- Level 9 is now a Class Capstone
- Level 10 is now a Subclass Capstone
